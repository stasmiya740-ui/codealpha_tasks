README 1 — Advanced To-Do List Web App
Advanced To-Do List Web App

A modern and responsive task management web application built using HTML, CSS, and JavaScript.
This app helps users organize daily tasks efficiently with advanced features like dark mode, category management, task filtering, drag-and-drop sorting, and progress tracking.

🚀 Features
Add, Edit, and Delete Tasks
Task Categories
Task Search Functionality
Completed & Pending Filters
Drag and Drop Task Reordering
Progress Tracking Bar
Dark/Light Mode Toggle
Toast Notifications
Local Storage Support
Fully Responsive Design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure

│
├── index.html
├── style.css
├── script.js
└── README.md
⚙️ How to Run the Project

Open the project folder
Run index.html in your browser
